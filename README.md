# 📺 多頻道監視器 - YT Live Monitor

一個充滿**賽博龐克 (Cyberpunk)** 與**毛玻璃 (Glassmorphism)** 科技風格的 YouTube 多頻道觀看儀表板。
只需輸入 YouTube 影片或直播網址，即可在同一個畫面中同時監控多達 9 個頻道，是觀看新聞直播、電競賽事、或 VTube 實況的完美「戰情中心」。

👉 **線上即時體驗：** [https://56webb.github.io/monimoni/](https://56webb.github.io/monimoni/)

---

## ✨ 核心特色功能

*   **⚡ 智慧多網格排版**
    *   內建 2、4、6、9 宮格的一鍵快速排版切換。
    *   純手工打造的動態 CSS 幾何圖示按鈕，懸停發出霓虹光暈。
*   **🚀 批次匯入引擎**
    *   支援一次貼上幾十個 YouTube 網址，系統自動過濾錯誤、剔除重複，並自動抓取頻道標題與縮圖。
*   **💾 群組記憶與一鍵切換**
    *   把當前畫面上的頻道組合儲存為專屬群組（如「新聞台」、「遊戲台」），點擊標籤即可一秒切換整個畫面。
*   **🔗 網址即時分享**
    *   目前的觀看畫面會自動同步到網址列。將網址傳給朋友，對方打開即可直接載入與你一模一樣的頻道組合！
*   **🔍 一鍵劇院放大模式 (Focus Mode)**
    *   滑鼠移至特定影片，點擊「放大」，該影片將瞬間填滿整個螢幕，不受其他頻道干擾；按下 `Esc` 即可退回九宮格。
*   **📱 頂級手機版體驗**
    *   在手機上自動轉為直式排列，保持 16:9 完美比例。
    *   獨創 **「防誤觸滑動遮罩 (Scrolling Shield)」**：影片上方 75% 覆蓋透明安全區，確保手機滑動網頁時絕對順暢，不會卡在影片內或誤觸暫停。
*   **📦 備份與同步**
    *   採用純前端 `LocalStorage` 技術，資料不外洩。
    *   內建「匯出/匯入」功能，自動將設定轉換為 Base64 代碼，輕鬆在不同電腦間轉移設定。

## 🛠️ 技術架構

*   **核心語法**：Vanilla HTML5, CSS3, JavaScript (ES6+)，完全零框架。
*   **外部 API**：使用 `https://noembed.com/embed` 輕量化取得 YouTube 標題與縮圖。
*   **嵌入技術**：YouTube IFrame Player API，並透過 `<meta name="referrer" content="unsafe-url">` 解決 Error 153 播放限制問題。
*   **部署環境**：GitHub Pages 靜態網頁託管。

## 🚀 如何在本地端執行

因為本專案為純靜態前端架構，你可以用最簡單的方式執行：

1. 將專案 Clone 或下載到你的電腦。
2. 直接雙擊打開 `index.html` 即可使用。
3. （可選）如果需要測試完整的網址列歷史紀錄功能，建議使用簡單的 HTTP Server 啟動，例如：
   ```bash
   python3 -m http.server 8080
   ```
   然後在瀏覽器前往 `http://localhost:8080` 觀看。
